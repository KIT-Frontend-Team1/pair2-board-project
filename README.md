# ⭐ pair2-board-project

### 1조 2페어의 게시판 토이 프로젝트 레퍼지토리입니다

## ➡️ 프로젝트 소개

😺 이미지 캐러셀, 페이지네이션, 게시물 및 댓글 CRUD 기능을 구현한 게시판 프로젝트

## ➡️ 프로젝트 개요

😺 faker.js 의 고양이 이미지를 활용한 게시글 형태(모달을 이용하여 게시물을 추가)의 메인 섹션과

페이지네이션을 이용한 인기 게시물 섹션으로 나누어 작업하였습니다.
